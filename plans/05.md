# Goal

Make every project consume the latest language-specific Git commit for its internal library dependency.

# Dependency graph

```text
once  ──depends-on──▶  big-config  ──depends-on──▶  selmer
```

# Plan

1. Resolve the current branch HEAD SHAs:
   - `bigconfig-ai/Selmer` branch `python`
   - `bigconfig-ai/Selmer` branch `typescript`
   - `bigconfig-ai/big-config` branch `clojure`
   - `bigconfig-ai/big-config` branch `python`
   - `bigconfig-ai/big-config` branch `typescript`

2. Update the direct dependency pins:
   - `big-config/python/pyproject.toml`
     - bump `selmer @ git+https://github.com/bigconfig-ai/Selmer.git@<sha>` to the latest `python` SHA
   - `big-config/typescript/package.json`
     - bump `selmer` from `github:bigconfig-ai/Selmer#<sha>` to the latest `typescript` SHA
   - `once/clojure/deps.edn`
     - bump `io.github.amiorin/big-config` `:git/sha` to the latest `clojure` SHA
   - `once/python/pyproject.toml`
     - bump `big-config @ git+https://github.com/bigconfig-ai/big-config.git@<sha>` to the latest `python` SHA
   - `once/typescript/package.json`
     - bump `big-config` from `github:bigconfig-ai/big-config#<sha>` to the latest `typescript` SHA

3. Refresh lock/resolution files after manifest edits:
   - `big-config/python/uv.lock`
   - `big-config/typescript/package-lock.json`
   - `once/python/uv.lock`
   - `once/typescript/package-lock.json`

4. Verify each touched project:
   - `big-config/python`: `uv sync --frozen && uv run pytest -q`
   - `big-config/typescript`: `npm run check && npm test`
   - `once/clojure`: `clojure -M:test`
   - `once/python`: `uv sync --frozen && uv run pytest -q`
   - `once/typescript`: `npm run typecheck && npm test`

5. Final checks:
   - Confirm no local path override is active where a Git coordinate is expected.
   - Confirm `.dist/` generated artifacts were not edited.
   - Review the diff to ensure only dependency pins and lockfiles changed.
